# Button

With `MDX`, we can define a story for `Button` right in the middle of our
Markdown documentation.

## Usage

```
import { Button } from '../components/button/button';

<Button label="" color="" variant="" size="" onClick="myFunction()" />
```

- medium is the secondary size and should cover most of the cases.
- Use large size in special cases to highlight the button.
